A python script to parse acquisition data from lights subfolders in a directory to use create an importable CSV for Astrobin acquisition data.

- Input constant acquisition values in the first section (gain, f-stop, sensor temperature, darks/biases/flats count, Bortle and SQM level).
- Input filters ID and name from Astrobin database in the second section (7 Touptek filters for S/H/O/L/R/G/B available by default).
- Run and verify CSV output.
- Import in Astrobin acquisition tab.
