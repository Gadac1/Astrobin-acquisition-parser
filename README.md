A python script to parse acquisition data from lights subfolders in a directory.

- Input constant acquisition values in the first section (gain, f-stop, sensor temperature, darks/biases/flats count, Bortle and SQM level).
- Input filters ID and name from Astrobin database in the second section (3 Touptek filters for S/H/O available by default).
- Run and verify CSV output.
- Import in Astrobin acquisition tab.
